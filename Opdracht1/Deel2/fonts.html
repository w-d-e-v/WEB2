Fonts in Bootstrap 5

Bootstrap 5 maakt het eenvoudig om lettertypen te beheren en aan te passen in je webprojecten. Het framework biedt standaard een set van systeemlettertypen, maar je kunt ook gemakkelijk aangepaste lettertypen toevoegen.

Standaard lettertypen

Bootstrap 5 gebruikt een reeks systeemlettertypen die automatisch worden toegepast op je tekst. De standaard lettertypen zijn:

font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol";

Aangepaste lettertypen

Als je aangepaste lettertypen wilt gebruiken, kun je deze eenvoudig toevoegen via CSS. Hier is een voorbeeld van hoe je een Google Font kunt toevoegen:

  1. Voeg de link naar het Google Font toe in de <head> van je HTML-document:
    2. <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
  2. Pas het lettertype toe op je elementen met CSS:
    3. body {
      font-family: 'Roboto', sans-serif;
    }

Lettergrootte en -stijl

Bootstrap 5 biedt ook klassen voor het aanpassen van de lettergrootte en -stijl. Hier zijn enkele voorbeelden:

  • Lettergrootte: Gebruik klassen zoals .fs-1 tot .fs-6 om de lettergrootte aan te passen.
  • Letterstijl: Gebruik klassen zoals .fw-bold voor vetgedrukte tekst en .fst-italic voor cursieve tekst.

Voorbeeld:

<p class="fs-1 fw-bold">Grote vetgedrukte tekst</p>
    <p class="fs-6 fst-italic">Kleine cursieve tekst</p>
